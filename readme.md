# Import Export Management System

China-to-India import/export business management. Next.js 14, TypeScript, MongoDB, shadcn/ui.

## Quick start

1. **Environment**  
   Copy `.env.local.example` to `.env.local` and set `MONGODB_URI` and `JWT_SECRET`.

2. **Install & run**
   ```bash
   npm install
   npm run dev
   ```

3. **Login**  
   Open http://localhost:3000 and sign in. Users are created by an Owner/Admin via the **Users** menu.
